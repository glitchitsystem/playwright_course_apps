  • Each Playwright test gets a fresh browser context by default
  • Browser contexts isolate cookies, localStorage, and sessionStorage
  • Use context.clearCookies() to reset authentication state
  • LocalStorage persists across page reloads within the same context
  • SessionStorage is cleared when the browser context closes
  • Create new contexts for testing different user sessions
